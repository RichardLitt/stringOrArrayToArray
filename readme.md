> Convert a String or an Array to an Array

## Install

```
$ npm install --save stringorarraytoarray
```

## Usage

```js
const toArray = require('stringorarraytoarray');

toArray('unicorns');
//=> ['unicorns']
toArray(['unicorns', 'rainbows']);
//=> ['unicorns', 'rainbows']
```

## API

### stringorarraytoarray(input)

#### input

Type: `string`, `array`
